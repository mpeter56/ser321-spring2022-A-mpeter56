You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html.
  • /json -- returns a json of the /random request.
  • /random -- returns index.html.
  • /multiply?num1='#'&num2='#' -- returns the product of num1 and num2.
  • /github?query=users/'username'/repos -- returns a list of repo owner names, ids, names
  • /color?color1='color' -- returns that color
  • /color?color1='color'&color2='color' -- returns a gradient of the two colors
  • /shape?shape='circle, square, rectangle, or ellipse'&color='color'&width='#(just use this for square and circle)'&height='#'
  • -- returns the shape with requested parameters and displays the area.
  • find colors at https://www.w3schools.com/colors/colors_names.asp
